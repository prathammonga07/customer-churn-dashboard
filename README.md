# Customer Churn Prediction Dashboard

A data analytics project predicting telecom customer churn 
using Logistic Regression with an interactive Streamlit dashboard.

## Key Findings
- 26.5% overall churn rate
- Month-to-month customers churn at 42%
- New customers (0-12 months) are highest risk
- Fibre optic customers churn at 41%

## Tech Stack
- Python, pandas, seaborn, scikit-learn
- Streamlit dashboard with AI risk explainer (Groq API)

## Project Structure
- churn_project1.ipynb — EDA + ML notebook
- app.py — Streamlit dashboard
